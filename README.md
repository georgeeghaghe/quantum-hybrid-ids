# Quantum-Enhanced Intrusion Detection System

![ROC Curve](results/roc_curve.png)

---

## Overview

This project explores how quantum computing concepts can support
cybersecurity intrusion detection using the CIC-IDS-2017 dataset.

The system combines:

- Classical machine learning
- Quantum-inspired anomaly detection
- Hybrid intrusion detection analysis

Built using Qiskit and Python.

---

## Features

- Quantum feature encoding
- Quantum probability analysis
- Jensen-Shannon anomaly scoring
- Hybrid classical + quantum detection
- ROC/AUC evaluation
- CIC-IDS-2017 traffic analysis

---

## Results

Both the classical and quantum-enhanced IDS models achieved strong
performance on CIC-IDS-2017.

This project demonstrates how quantum computing concepts can integrate
into future cybersecurity pipelines.

---

## Quantum Design

The quantum circuit uses:

- Ry rotation gates
- CNOT entanglement
- Rz phase gates
- Quantum probability distributions

to model network traffic behavior.
